# Add Favorites Management Page for Customers

Add a new page in the customer section called "Favorites" where customers can manage their favorite salons. This is critical for customer loyalty because men have favorite barbers and women have favorite salons.

## What to Build:

1. Create a new tab in the bottom navigation called "Favorites" (❤️ icon) between "Bookings" and "Profile"

2. Favorites Page should show:
   - List of all favorite salons (each as a card)
   - Each card shows: salon name, rating (stars), distance, live availability ("3 chairs available"), favorite barber name, wait time ("5 min"), and buttons [Book Now], [View Map], [Remove]
   - Filter buttons at top: All, Men's Barbers, Women's Salons, Near Me, Available Now
   - Search bar to search by salon name, barber name, or city
   - Empty state if no favorites: "No favorite salons yet. Tap ❤️ on any salon to add" with [Browse Salons] button
   - Pull-to-refresh to reload favorites

3. Add heart button (❤️) to all salon cards in the app:
   - Tap heart to add/remove from favorites
   - Heart is red when favorite, white when not
   - Show notification "Added to favorites!" when added

4. Backend API to add:
   - GET /api/users/favorites (get user's favorite salons)
   - POST /api/users/favorites/:salonId (add to favorites)
   - DELETE /api/users/favorites/:salonId (remove from favorites)
   - GET /api/users/favorites/check/:salonId (check if favorite)

5. Database: Add Favorite model with userId, salonId, createdAt (unique constraint on userId + salonId)

6. Mobile components to create:
   - FavoritesTab.tsx (new tab screen)
   - useFavorites.ts (hook for fetching/adding/removing favorites)
   - Update SalonCard.tsx to add heart button and live availability badge

7. Analytics to track:
   - favorite_added (when user adds salon)
   - favorite_removed (when user removes salon)
   - favorites_viewed (when user opens favorites tab)
   - quick_book_from_favorite (when user books from favorites)

Why this matters: Customer loyalty is critical in grooming. Men remember favorite barbers, women remember favorite salons. This feature increases retention by 3-5x and makes booking 5x faster (1 tap instead of searching).
